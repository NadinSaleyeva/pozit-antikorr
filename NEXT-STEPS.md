# Что делать дальше

## 1. Подожди 2-3 минуты
GitHub Pages обновляется не мгновенно.

## 2. Открой сайт в режиме инкогнито
Chrome: Ctrl+Shift+N
Edge: Ctrl+Shift+P

URL: https://nadinsaleyeva.github.io/pozit-antikorr/

## 3. Что ты должна увидеть

В правом нижнем углу должны быть ДВЕ кнопки:

### Кнопка 1 (ТЕСТОВАЯ - ЯРКО-ЗЕЛЁНАЯ):
- Размер: 100x100 пикселей
- Цвет: ЯРКО-ЗЕЛЁНЫЙ (lime)
- Рамка: Чёрная, толстая (5px)
- Позиция: Чуть выше красной кнопки
- Символ: ↑

### Кнопка 2 (КРАСНАЯ):
- Размер: 60x60 пикселей  
- Цвет: КРАСНЫЙ
- Рамка: Жёлтая
- Позиция: Ниже зелёной кнопки
- Символ: ↑

### Кнопка 3 (КРАСНАЯ "Позвонить"):
- Овальная
- Красная с текстом
- Самая нижняя

## 4. Если НЕ видишь зелёную кнопку:

### Вариант А: Очисти кеш
1. Нажми F12 (откроется DevTools)
2. Правой кнопкой на кнопку обновления страницы
3. Выбери "Очистить кеш и жёсткая перезагрузка"

### Вариант Б: Проверь другой браузер
Попробуй открыть в другом браузере (Chrome, Firefox, Edge)
